版本依赖信息   


|序号  |软件 | 版本   |  
| ------ | ------ |------ |  
|01|java | 1.8 |  
|02|spring | 5.1.5.RELEASE  |  
|03|spring boot | 2.1.3.RELEASE|  
|04|google guava | 18.0 |  
|05|lombok | 1.18.6 |  
|06|mysql | 5.x |  

运行方式：  
编译方式：mvn clean install -DskipTests  

启动程序：java -jar personal-website-blogs.jar
